# ScholarLib

A professional academic reference manager with AI-powered document Q&A.

**Live App:** https://pourmousavi.github.io/scholarlib

## Features

- PDF viewing and annotation
- AI-powered document Q&A (local via Ollama/WebLLM or cloud via Claude/OpenAI)
- University Box / Dropbox integration for PDF storage
- Folder organization and tagging
- Cross-platform (Mac, Windows, iPad)

## Development

```bash
npm install
npm run dev
```
