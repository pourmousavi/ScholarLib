# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in ScholarLib.

## Navigation

| Shortcut | Action |
|----------|--------|
| <kbd>↑</kbd> <kbd>↓</kbd> | Navigate document list |
| <kbd>Enter</kbd> | Open selected document |
| <kbd>Esc</kbd> | Close modal / Clear search |

## AI Chat

| Shortcut | Action |
|----------|--------|
| <kbd>Enter</kbd> | Send message |
| <kbd>Shift</kbd> + <kbd>Enter</kbd> | New line in message |

## General

| Shortcut | Action |
|----------|--------|
| <kbd>Cmd/Ctrl</kbd> + <kbd>K</kbd> | Focus search |
| <kbd>Cmd/Ctrl</kbd> + <kbd>,</kbd> | Open settings |

:::tip Pro Tip
Most modal dialogs can be closed by pressing <kbd>Esc</kbd>.
:::
