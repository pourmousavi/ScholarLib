# Getting Started

Welcome to ScholarLib, a professional academic reference manager that keeps your PDFs private in your own cloud storage while providing powerful AI-assisted research capabilities.

## Quick Start Checklist

1. **Connect Storage** - Link your Dropbox or Box account to store PDFs securely in your own cloud.

2. **Set Up AI** - Choose an AI provider: WebLLM (free, in-browser), Ollama (local), or cloud APIs.

3. **Create Folders** - Organize your library with folders. Right-click the sidebar to create new folders.

4. **Upload PDFs** - Drag & drop PDFs into a folder. Metadata is automatically extracted.

5. **Start Chatting** - Select a document and use AI Chat to ask questions about your papers.

## Key Features

- **Private Storage** — Your PDFs stay in your Dropbox/Box, never on our servers
- **Smart Metadata** — Automatic extraction via GROBID, CrossRef, and AI
- **Flexible Organization** — Folders for storage, tags for topics, collections for projects
- **AI Chat** — Ask questions scoped to documents, folders, tags, or collections
- **Citation Export** — Export to BibTeX, RIS, APA, MLA, Chicago, and more
- **Collaboration** — Share folders, tags, or collections with others
- **Notes** — Take notes on papers with export to Markdown, PDF, or Word
- **PWA Support** — Install as an app on Mac, Windows, or iPad

:::tip Pro Tip
Use keyboard shortcuts for faster navigation. Press <kbd>Cmd/Ctrl</kbd> + <kbd>K</kbd> to quickly search your library. See the [Keyboard Shortcuts](/shortcuts) page for all available shortcuts.
:::
